DEAD: Box Model Homepage

The box model refers to the stand-alone version of the DEAD aerosol model. The box model is driven by a standalone program which supplies all the information normally input from the host GCM or CTM. The user interactively specifies the number of timesteps and the environmental conditions to the box model. The script which allow the box model to run interactively in a web server environment are also available for downloading and modification.
  • Box model web-scripts in tarball format: bxm.tar.gz
    • We encourage users to examine and to send us improvements to the current scripts. If there are particular variables or NCL graphics techniques that you desire, simply modify our existing scripts and send us the results. We are happy to incorporate useful new features of general interest.
